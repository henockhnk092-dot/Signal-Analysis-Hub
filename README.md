## Project Scope
This project focuses on Engineering Mathematics, Signal Processing, and Dynamic System Modeling. The content is designed to provide step-by-step visual guides for graphing, reasoning about signals, and understanding core math foundations used in system analysis.

## Core Subject Areas
- Signal analysis and modeling: unit step, unit impulse (delta), ramp, continuous-time and discrete-time signals, and Laplace transform context.
- Calculus and algebra foundations: parent functions, tangent lines, and coordinate systems.
- Function properties and waveform analysis: periodicity, even and odd symmetry, and piecewise representations.

## Studio Features
- Lesson Library with three learning tracks: Fundamental Signals, Signal Transformations, Periodic Waveforms.
- Graph Studio tools: signal plotter, equation plotter, draw-to-equation, and calculator.

## Key Concepts Map
| Category | Key Concepts | Equations and Notations |
| --- | --- | --- |
| Basic Signals | Impulse, Step, Ramp, Parabolic | 1/s, 1/s^2, 1/s^3 (Laplace) |
| Transformations | Shifting, Scaling, Reversal | f(t +/- k), A * f(t), f(-t) |
| Periodicity | Cycle length, repetition | f(t) = f(t + T) |
| Symmetry | Even (y-axis), Odd (origin) | f(-t) = f(t) or f(-t) = -f(t) |
| Foundations | Cartesian plane, Parent functions | (x, y) coordinates, y = mx + c |
